# Кэшач0к — конвертер

Готовая статичная страница для GitHub Pages.

## Структура

- `index.html` — разметка страницы
- `styles.css` — стили и адаптив
- `script.js` — логика конвертера и бургер-меню
- `images/` — папка для будущих PNG/SVG

## Какие файлы положить в `images/`

- `logo.svg`
- `doshik.png`
- `zucchini.png`
- `coffee.png`
- `banana.png`
- `mandarin.png`

Если иконок пока нет, страница покажет emoji-заглушки.

## Публикация на GitHub Pages

1. Создать репозиторий.
2. Загрузить файлы в корень репозитория.
3. В настройках репозитория открыть `Pages`.
4. Выбрать ветку `main` и папку `/root`.
5. Сохранить.
